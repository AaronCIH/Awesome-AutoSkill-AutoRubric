# Contributing to Awesome AutoSkill & AutoRubric

Thank you for your interest in contributing!

## How to Add a Paper

1. **Fork** this repository
2. **Edit** `README.md` and add your entry to the appropriate section
3. **Follow the format**:

   ```
   | **Paper Name** | Month Year | [arXiv:XXXX.XXXXX](https://arxiv.org/abs/XXXX.XXXXX) | [Code](https://github.com/...) or `-` | One-sentence TL;DR |
   ```

4. **Submit** a Pull Request with a clear description

## What to Include

- **Auto-Skill papers**: Research on self-evolving agent skills, skill creation, skill reuse, skill libraries, skill evolution
- **Auto-Rubric papers**: Research on automated rubric discovery, rubric-based reward modeling, rubric learning from preferences/comparisons
- **Related work**: Closely related papers on agent self-evolution, memory-based self-improvement, etc.

## Quality Standards

- Papers should be peer-reviewed or on arXiv
- Include working links to papers and code repositories
- TL;DR should be concise (1-2 sentences) and informative
- Sort entries chronologically within each section
- Note any venue acceptance (e.g., ICML 2026, ACL 2026)

## Code of Conduct

Be respectful and constructive in all interactions. We welcome contributions from everyone.
